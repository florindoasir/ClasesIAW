# Ejemplo: almacenar en un fichero la variable GET
```PHP
<?php

// http://localhost:8000/?valor=holadf

	$valor = $_GET["valor"];

	$file = fopen("archivo.txt", "w");
	fwrite($file, $valor . PHP_EOL);
	fclose($file);

	$file = fopen("archivo.txt", "r");
	
	while(!feof($file)) {
		echo fgets($file). "<br />";
	}

	fclose($file);
?>
```
