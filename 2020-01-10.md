# Ejemplos

## Escribir y leer información en un fichero en PHP

```PHP
<?php
	$file = fopen("archivo.txt", "w");
	fwrite($file, "Contenido del fichero" . PHP_EOL);
	fclose($file);

	$file = fopen("archivo.txt", "r");
	
	while(!feof($file)) {
		echo fgets($file). "<br />";
	}

	fclose($file);
?>
```
